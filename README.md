# 2048

To play the game, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/2048.git
   ```

2. Navigate to the project directory:
   ```
   cd 2048
   ```

3. Compile the project:
   ```
   mvn package
   ```

4. Run the game:
   ```
   java -jar target/2048.jar
   ```
